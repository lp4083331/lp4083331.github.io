# 题目

已知链表头节点指针head，将链表逆序。（不可申请额外空间）

# 代码实现

## C语言

```
Node* link_reverse(Node *head)
{
    Node *new_head = NULL;
    Node *next = NULL;

    while (head != NULL)
    {
        next = head->next;
        head->next = new_head;
        new_head = head;
        head = next;
   }

    return new_head;
}
```



## Go语言

```
func LinkReverse(Head *Node) (*Node) {
	var NewHead *Node
	for(Head != nil) {
		next := Head.next
		Head.next = NewHead
		NewHead = Head
		Head = next
	}
	return NewHead
}
```
